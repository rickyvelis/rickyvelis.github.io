Delirium

                
Six other students and I developed Delirium,
                  which is a randomly generated horror-puzzle game made in Unity3D during the 2021 Game
                  Development & Simulation minor at The Hague University of Applied Sciences.
                  


                  I was mainly responsible for designing and programming the behaviour of the stalking enemy AI.
                  My work also included modeling and placing 3D assets, contributing to designing the game's mechanics and documenting them in a game design document.

Cyber Intrusion Survival Game

                

                  Together with a group of other students, I developed an educational game concerning various cyber threats that companies have to deal with.
                  This game simulates a custom OS inside that is used inside financial company. Inside this OS, the player has to perform various tasks in order to handle different cyber threats, like a ddos attack, a ransomware
                  During the development of this game I gained a lot of knowledge about UI development for Unity, as the game is mostly UI based.
                  


                  This game is part of the Dutch Joint Cyber Range initiative; a collaboration among
                  multiple educational institutes in the Netherlands, to provide a platform for technical security
                  education.
